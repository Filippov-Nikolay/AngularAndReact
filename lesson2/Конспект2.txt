Создание проекта: ng new my-first-project
Переход в директорию с проектом: cd my-first-project
Запуск: ng serve/npm start
Установка модулей(локально для проекта): npm install
Без файла package.json в структуре проекта, модули node не устанавливаются
angular.json - не рекомендуется менять связь! (Настройки ангуляра)

Компоненты в Angular:
		Компонент - это совокупность тега, шаблона и css файла + TS класса. Т.е. минимальная единица Angular это компонент

</> Компоненты в Angular создаются с помощью декоратора @Compontent.
</> Декоратор - функция которая динамично подключает дополнительные возможности и поведение к объекту, расширяя его функциональность.
</> Компонент - конструкция которая объединяется по общему признаку в повторном использовании и управляет отображением представления на экране. Свойства компонента:
   * selector - определяет тег, для подключения этого компонента
   * standalone - если true, то объект является автономным
   * imports - массив модулей который будут использованы в этом компоненте
   * templateUrl - файл с разметкой компонента
   * styleUrl - css файл компонента
